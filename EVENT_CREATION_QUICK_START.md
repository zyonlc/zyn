- Toggle on if yes
- Paste your livestream link (YouTube, etc.)
- Viewers will see "Livestream available" badge on event card

## â“ Frequently Asked Questions

**Q: Can I edit my event after creating it?**
A: Yes! Publish functionality will be added soon for editing. For now, you can delete and recreate if needed.

**Q: Are service providers automatically confirmed?**
A: No, they receive your booking request with status "pending". They'll respond within 24 hours.

**Q: Can I book services after publishing?**
A: Yes! You can add services anytime by clicking "Add Services" in My Events.

**Q: Is there a cost to create an event?**
A: No! Creating events is free. You only pay when booking service providers.

**Q: Can other people see my draft events?**
A: No, only published events are visible to others. Draft events are private.

**Q: Can I have multiple events?**
A: Yes! Create as many events as you want. Each appears in your My Events list.

**Q: What if I want to cancel an event?**
A: Simply delete it from My Events. You may need to notify booked providers separately.
